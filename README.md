## Run
- fill the .env file taking into account .env.example
- and run
```bash
npm run build
source .env && node .output/server/index.mjs
```

## Setup

```bash
# npm
npm install
```

## Development

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
